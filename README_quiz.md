# Quiz de Capacitación (Streamlit + SQLite)

## Requisitos
- Python 3.9+
- `pip install streamlit pandas`

## Ejecutar
```bash
streamlit run quiz_app.py
```

## Notas
- PIN de administrador por defecto: `4321` (cámbielo con variable de entorno `ADMIN_PIN` o editando el archivo).
